# SmartFork v2 — Obsidian Vault Export (Layer 9)

> **Design:** Auto-generate an Obsidian-compatible knowledge vault from indexed sessions.
> **Output:** Directory with MOC, session notes, Mermaid graphs, Dataview queries.
> **Command:** `smartfork vault --output ./my-vault`

---

## Vault Structure

```
my-vault/
├── MOC.md                    # Map of Content with Mermaid graph + Dataview
├── _Graph/
│   └── Graph View.md         # Wiki-links for Obsidian's native graph view
├── Sessions/
│   ├── <session_id[:16]>.md  # One note per session
│   └── ...
├── Projects/
│   ├── <project_name>.md     # Per-project MOC (optional, --project-folders)
│   └── ...
└── .obsidian/
    ├── app.json              # Obsidian config
    └── appearance.json        # Theme settings
```

## Session Notes

**File:** `Sessions/<id[:16]>.md`

Each session note has YAML frontmatter + structured body:

```yaml
---
tags:
  - session
  - <project_tag>        # sanitized: lowercase, hyphens
  - status/<status>      # superseding | superseded | none
aliases:
  - "<session_id[:12]>..."
supersession_status: superseding | superseded | none
project: "<project_name>"
duration: <float_minutes>
start_time: "<YYYY-MM-DD HH:MM>"
end_time: "<YYYY-MM-DD HH:MM>"
session_id: "<full_id>"
agent: "<agent_id>"
domains: ['domain1', 'domain2']
languages: ['python', 'typescript']
quality_tag: solution_found | dead_end | partial | reference | unknown
tech_tags: ['FastAPI', 'JWT']
model_used: "qwen2.5-coder:7b"
---
```

**Body sections:**

```markdown
# Session: <title or task[:80]>

**Session ID:** `<full_id>`
**Project:** [[<project_name>]]
**Agent:** <agent_display_name>
**Status:** <supersession_status>
**Duration:** <minutes> min
**Quality:** <quality_badge>

## Task
> <task_raw[:500]>

## Summary
> <summary_doc>

## Technologies
**Domains:** <domains>
**Languages:** <languages>
**Frameworks:** <tech_tags>

## Timeline
- **Started:** <start_time>
- **Ended:** <end_time>

## Files Modified
<files_edited as comma list>

## Key Insights
> <first 2 reasoning blocks, truncated 300 chars each>

## Supersession
### Supersedes
- [[<session_id[:16]>]]

### Superseded By
- [[<session_id[:16]>]]

## Related Sessions
\`\`\`dataview
TABLE duration, project, supersession_status
FROM #<project_tag>
WHERE contains(file.inlinks, this.file.link) OR contains(file.outlinks, this.file.link)
  AND session_id != "<this_id>"
\`\`\`
```

## YAML Frontmatter Safety

- Escape double quotes in string values
- Sanitize project names: lowercase, replace spaces/special chars with hyphens
- Truncate long task descriptions in Mermaid labels
- Handle empty fields gracefully (skip, don't error)
